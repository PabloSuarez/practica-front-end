Sobre mi

Soy un joven programador con 4 años de experiencia que me han permitido trabajar en sistemas varios como
Erp`s, de logística, financieros, producción(fábrica) y por último para una plataforma de Virtual Banking.

Deseo dedicarme al desarrollo web preferentemente pero no descarto trabajar como backend y demas opciones,
me gusta mucho el desarrollo y rápidamente me adapto a la situación.

He trabajado en desarrollo web con: git, bitbucker, Html, css, js, jquery, jsp, struts, java, maven, sql.

Para sistemas Windows Mobile: C# y Web services.

Para sistemas de escritorio sobre: Java, clarion y c#.

También poseo experiencia como freelance trabajando de scraper sobre anuncios web con Php como lenguaje.

3 mejoras en html5

Se han incorporado nuevas etiquetas como < video > < audio > que permiten de manera muy amena reproducir contenidos.

Se incluyen controles como el validaror de correo, o de urls que ya no requieren de js, permitiendo con css informar al usuario si es válido o no el valor ingresado.

Se agregan localStorage y sessionStorage dando posibilidad de almacenacimento en el cliente, teniendo mucha mas capacidad que una cookie y siendo muy sencillas de utilizar.

3 mejoras en css3

Por ejemplo se agregan las etiquetas para border-color, border-radius (utilizadas en esta web, en imagen de perfil).

Se agregan las media queries para detectar la resolucion de la pantalla, utilizadas para hacer responsivo este sitio.

Flexbox que llego para potenciar y facilitar el responsive design, ya no es necesario float para esto.
Es muy facil de usar y tiene muchas opciones personalizables.

Javascript: ejemplo con Google Maps

3 técnicas para optimizar perfomance

La compresion de html, css y js, para esto se puede usar stylus, jade y babel o algun gestor de tareas como Gulp y sus respectivos plugins.

Reducir la menor cantidad las llamadas al servidor, es recomendado tener un solo archivo para css, js y html respectivamente.
También existe la posibilidad de reducir la llamada a imágenes en el servidor con los mapas de imágenes.

El uso de cache puede ser muy beneficioso, actualmente disponemos de localStorage también.
